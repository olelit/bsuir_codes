Цель работы.

В настоящей работе необходимо познакомиться с рисованием пером в С#. Надлежит построить график синусоиды или другой математической функции.

Краткие теоретические сведения.

Для рисования линий и фигур нужно получить сначала графический контекст. Затем выдать команды рисования. Следующий фрагмент поясняет сказанное:

Graphics g= e.Graphics;

                   Pen p = new Pen(Color.Bisque,5);

                   g.DrawRectangle(p,10,10,200,200);

Графическому контексту передается обработчик события OnPaint. Поэтому можно привязаться именно к этому обработчику. Смысл наших действий таков: поставить кнопку и закрепить за ней обработчик события OnClick. В этом пункте обработчику следует просто вызвать перерисовку формы, где и получить фигуры. Это делается так:

private void button1_Click(object sender, System.EventArgs e)

                {

                          show=1;

                          this.Invalidate();

                }

Мы намеренно используем переменную show для того, чтобы отменить рисование, когда show=0. Метод Invalidate как раз и активизирует событие OnPaint.

Собственные обработчики событий закрепляются за событиями в конструкторе:

this.button2.Click += new System.EventHandler(this.button2_Click);

Эта команда закрепляет программу  button2_Click  за событием нажатия на кнопку.

this.Paint += new System.Windows.Forms.PaintEventHandler(this.showgraph);

Аналогичным образом закрепляет программу    в качестве обработчика события OnPaint.

Другой способ использования графического контекста - это прямое его создание следующим образом:

Graphics g = control.CreateGraphics();

Здесь в качестве control может выступать форма.
