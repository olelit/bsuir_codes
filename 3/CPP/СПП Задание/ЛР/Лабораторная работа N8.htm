Краткие теоретические сведения

Нас будет интересовать задача построения почтового клиента средствами языка C#.  Почтовые клиенты могут использовать два основных типа протокола – SMTP и POP3. Протокол SMTP (Simple Mail Transfer Protocol) использует следующие команды:

HELLO  – команда инициации связи

RECIPIENT – получатель сообщения

DATA – начало постовых данных

SEND  – инициирует транзакцию для отправки данных

RESET – аварийно завершает текущую почтовыю транзакцию

QUIT – команда для закрытия соединения с почтовым сервером

Сервер возвращает в ответ на команды клиента коды завершения операций. Эти коды представляют трехзначные числа. Первая цифра имеет следующие значения:

1-    ожидание подтверждения

2-    успешное завершение команды

3-    ожидание дополнительной информации

4-    неудачное завершение (временное состояние)

5-    неудачное итоговое завершение

Вторая цифра

0 – ошибка синтаксиса команды

1-    успешное соединение

5-почта
